# B3bas Uploader

B3bas Uploader is simple S3 uploader file, build on the top of B3bas Golang Framework.

## Changelog
* Changing documentation in this [file](./CHANGELOG.md).

## How To
* Build (using `make`)
  ```
  make build
  ```
* Build (using `go` binary)
  ```  
  go build app.go
  ```

* Run (using `make`)
  ```
  make run
  ```
* Run (using `go` binary)
  ```
  go run app.go
  ```
